# \# 视频倒放工具 (Python3.13 兼容版)

# 一款\*\*无需安装环境、双击即用\*\*的桌面工具，支持视频与音频同步倒放，适配 Windows 系统，完美兼容 Python 3.13。

# 

# \## 功能亮点

# ✅ \*\*音视频同步\*\*：一键实现视频画面与音频的同步倒放，无延迟、不同步问题  

# ✅ \*\*零环境依赖\*\*：无需安装 Python、FFmpeg 等任何组件，下载即用  

# ✅ \*\*格式兼容\*\*：支持 MP4、AVI、MOV、MKV 等主流视频格式  

# ✅ \*\*操作简单\*\*：可视化界面，仅需选择视频与保存路径，点击即可完成处理  

# 

# \## 快速使用（用户必看）

# \### 方式一：单文件版（推荐）

# 1\. 下载仓库发布的 `ReverseVedio.exe` 单文件  

# 2\. \*\*双击直接运行\*\*，无需任何额外文件  

# 3\. 按照界面提示：选择待倒放视频 → 选择保存文件夹 → 点击「开始倒放」  

# 4\. 等待处理完成，在保存路径中找到后缀为「\_倒放」的视频文件  

# 

# \### 方式二：双文件版

# 若获取的是分离文件，请将 `ReverseVedio.exe` 与 `ffmpeg.exe` 放在\*\*同一文件夹\*\*，再双击 `ReverseVedio.exe` 运行。
